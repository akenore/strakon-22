Système d'exploitation/matériel recommandé pour STRAKON

Windows 10, 8.x

Utilisation de STRAKON 2D

  • Pas de demandes accrues sur le processeur, la carte graphique et la mémoire
  • Deux écrans, c'est l'idéal

STRAKON 3D/BIM utilisation supplémentaire

  • Optimal est un bon processeur à partir de 4 GHz avec plusieurs cœurs de calcul
  • Carte graphique moyenne "Gaming" (OpenGL 4.0 ou supérieur)
  • 32 Go de mémoire

STRAKON pour le bureau à distance

Si vous voulez utiliser STRAKON via un bureau à distance (travail à partir de votre bureau à domicile sur l'ordinateur de votre entreprise), vous avez besoin d'un dongle réseau. En outre, toutes les cartes graphiques ne fonctionneront pas avec STRAKON en liaison avec Open GL (uniquement pour l'utilisation de STRAKON-Cube 3D) :
  • Les cartes graphiques NVIDIA GeForce ne fonctionnent pas.
  • Les cartes graphiques NVIDIA Quadro fonctionnent.
  • Avec les cartes AMD, l'image n'est pas uniforme jusqu'à présent. En cas de doute, il convient de consulter le revendeur de matériel.

STRAKON pour une utilisation de bureau à distance sur un serveur avec différents profils d'utilisateurs

STRAKON est exécutable en tant que RemoteApp à partir de Windows Server 2016 Remote Desktop Services (RDS). (STRAKON doit être exploité avec une licence réseau.) Les performances 3D dépendent du matériel utilisé. Les Remote Desktop Services font partie intégrante de Microsoft Server 2016 et supérieur (avec la licence appropriée).

La barre d'outils n'est pas disponible

Question

Dans STRAKON, le réticule ne peut pas être déplacé vers la barre d'outils avec le codeur de coordonnées Wacom après avoir changé le moniteur ou la résolution de l'écran.

Réponse

  • Fermer STRAKON
  • Démarrer > Programmes > Wacom Tablet > Wacom Tablet Preferences File Utility > Mes préférences > supprimer
  • Démarrer > Tous les programmes > Wacom Tablet > Wacom Tablet Properties
    • Placez le stylo sur la tablette et reconfigurez les boutons du stylo comme suit :
    • pointe du stylo > désactivé
    • bouton à bascule inférieur > cliquer
    • bouton à bascule supérieur > clic droit de la souris

Votre temps d'utilisation expirera dans quelques jours

Question

Au démarrage de STRAKON, le message Votre temps d'utilisation a expiré dans quelques jours s'affiche.

Réponse

Via Démarrer > Tous les programmes > DICAD > Dongletest, vérifiez si la version actuelle, par exemple la version 20xx, est répertoriée derrière le numéro de version STRAKON. Si c'est le cas, vous pouvez continuer à travailler sans problème. Après l'expiration du délai d'utilisation, le message n'est plus affiché.
Si le numéro de la version actuelle n'est pas indiqué, veuillez contacter votre représentant commercial.

Crosshair dessine une piste

Question

Lorsque le réticule est déplacé, une piste est tracée.

Réponse

  • Fermer STRAKON
  • Ouvrez les paramètres graphiques de Windows et ajustez la taille de la police à la valeur par défaut de 96 dpi ou 100%.
